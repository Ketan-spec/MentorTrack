# MentorTrack  
### A smart web app that streamlines mentorship interactions and progress tracking.

MentorTrack is a full-stack web application built using **Flask**, **SQLite**, **HTML**, **CSS**, and **JavaScript**.  
It helps mentors and mentees collaborate efficiently through role-based dashboards, task management, and progress monitoring.
